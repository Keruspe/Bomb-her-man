Installation de Bomb-Her-Man

Tout d'abord, il vous faudra installer les paquets suivants sur votre système avec votre gestionnaire de paquets préféré : SDL_ttf, cairo et rsvg.

Ensuite, il faut décompresser l'archive du jeu (tar -xf Bomb-her-man.tar.gz) puis se rendre dans le dossier décompressé (cd Bomb-her-man/).

Il suffit alors de lancer le make et make install en root pour réaliser la compilation et l'installation (make && sudo make install).

Vous pouvez à ce stade supprimer le tar.gz du jeu ainsi que le dossier décompressé (cd ..; rm -rf Bomb-her-man/; rm Bomb-her-man.tar.gz)

Le jeu est maintenant accessible dans votre dossier bin local (/usr/local/usr/bin/bomb-her-man). Have fun :-)

Déroulement d'une partie
Une fois la partie lancée, les règles sont simples : le dernier en vie a gagné. Pour tuer son adversaire, il faut que celui-ci se trouve pris dans le rayon d'explosion d'une bombe (la votre, la sienne, ou celle d'un ordinateur, peu importe).
Les bombes ont, avec les paramètres de base, une portée de 2. C'est à dire qu'elles explosent sur une croix comprenant la case de la bombe et deux cases dans chaque direction (pas d'explosion en diagonale) en s' arrêtant toutefois si elle recontre une tombe sur son passage (celles-ci étant indestructibles). De plus et toujours avec les paramètres de base, vous pouvez poser jusqu'à trois bombes sur le terrain.
